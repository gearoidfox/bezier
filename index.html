A Bézier curve is a type parametric curve, specified by a set of control points. An n'th order Bézier curve has n+1 control points P, and can be defined as:

This means that for each value of t, B(t) is a weighted average of the control points P. As t varies from 0 to 1, the weight of each control point changes, causing B(t) to trace out a smooth curve.

This pattern continues for higher order curves, with a Bézier curve of order n being a linear interpolation between two curves of order n-1. If the n'th order curve has control points P[0]..P[n], the two lower order curves have control points P[0]..P[n-1] and P[1]..P[n], respectively.
